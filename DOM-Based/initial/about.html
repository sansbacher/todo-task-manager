Todo Task Manager

Manage all you have Todo!

The Vanilla JavaScript DOM-based Todo Task Manager!

It seems everyone writes a Todo / Task app when learning Web/HTML/CSS/JavaScript - this is one of those!

In fact it's one of three variations of the Todo / Task app. This one uses the DOM itself to save the 'state' - if you refresh the page all your Tasks will be gone. The page's DOM is directly manipulated. It's all client-side JavaScript.

It's loosely based on one of many similar DOM-basede Todo apps out there. Hopefully it's useful to someone else.

The other versions of the Todo / Task app use Browser LocalStorage or interface to a Back-End REST API - in those cases the 'state' is stored somewhere and manipulated in JavaScript. The page's DOM just reflects the current 'state'.

I'm less interested in graphic design or fancy web page aesthetics or really much of the whole front-end HTML/CSS than I am interested in the back-end API or general JavaScript. So I just wanted something that looked alright and would work. I picked W3.CSS as it's free, simple, and well documented. I just needed to know enough HTML and CSS to modify the pages to work for each variation. Sorry if the page design isn't very exciting.
